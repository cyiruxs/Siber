| Protocol                                       | Transport Protocol | Default Port Number |
| ---------------------------------------------- | ------------------ | ------------------- |
| TELNET                                         | TCP                | 23                  |
| DNS (**Domain Name System**)                   | UDP or TCP         | 53                  |
| HTTP (**Hypertext Transfer Protocol**)         | TCP                | 80                  |
| HTTPS (**Hypertext Transfer Protocol Secure**) | TCP                | 443                 |
| FTP (**File Transfer Protocol**)               | TCP                | 21                  |
| SMTP (**Simple Mail Transfer Protocol**)       | TCP                | 25                  |
| POP3 (**Post Office Protocol Version 3**)      | TCP                | 110                 |
| IMAP (**Internet Message Access Protocol**)    | TCP                | 143                 |

### Insecure Versions (Default TCP Ports)

| Protocol | Default Port Number |
|----------|---------------------|
| HTTP     | 80                  |
| SMTP     | 25                  |
| POP3     | 110                 |
| IMAP     | 143                 |

### Secure Versions (TLS)

| Protocol | Default Port Number |
|----------|---------------------|
| HTTPS    | 443                 |
| SMTPS   | 465, 587            |
| POP3S   | 995                 |
| IMAPS   | 993                 |
